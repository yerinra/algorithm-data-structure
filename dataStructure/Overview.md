# 알고리즘 복잡도 (시간 복잡도)

입력 크기의 값에 대해 단위 연산을 몇 번 수행하는지 계산하여 알고리즘의 수행시간을 평가하는 방법.

##3가지 점근적 표현법

- O(빅오): 최악의 상황을 고려하여 성능 측정 결과 표현.
- 세타 : 평균적인 경우에서의 성능 측정 결과 표현
- 오메가 : 최선의 상황일 때의 성능 측정 결과 표현

# 알고리즘 평가 지표

1. 정확성
2. 작업량
3. 메모리 사용량
4. 최적성
5. 효율성
